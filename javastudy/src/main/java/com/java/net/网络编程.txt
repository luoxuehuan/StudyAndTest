网络结构，
1,C/S  client/server
	
	特点：
		该结构的软件，客户端和服务端都需要编写。
		可发成本较高，维护较为麻烦。
		
	好处：
		客户端在本地可以分担一部分运算。




2,B/S  browser/server
	特点：
		该结构的软件，只开发服务器端，不开发客户端，因为客户端直接由浏览器取代。 
		开发成本相对低，维护更为简单。
	缺点：所有运算都要在服务端完成。
